# Collaboration Demo Project

This project is used to practice Git and GitHub collaboration workflows.

## How to contribute
- Open an issue detailing your proposed change.
- Create a new branch using naming conventions (e.g., `feature-*` or `docs-*`).
- Submit a pull request and link it to your issue (e.g., using `Closes #1`).
- Ensure the CI pipeline passes successfully (fix any errors before requesting a review) and wait for code review before merging.
